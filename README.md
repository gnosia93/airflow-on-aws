# airflow-on-aws

이번 워크샵에서는 에어플로우를 이용하여 피처 처리 등의 변환과 알고리즘 학습 및 예측을 한번에 실행하기에 필요한 여러가지 기술 요소에 대해서 설명합니다. 
일반적으로 머신러닝 모델을 구축하는 주요 프로세스는 피처의 가공, 변경, 추출을 수행하는 피처 처리, ML 알고리즘 학습 및 예측 수행, 그리고 모델 평가 단계를 반복적으로 
수행하는 것입니다.
